# protein visualization gallery

Code to generate the 3D structure gallery embedded in [this pub](https://doi.org/10.57844/arcadia-2u60-81sg).

## Overview

`generate_gallery.py` reads `gallery_config.json`, downloads the `model-viewer` JS library and web font, and produces a single self-contained `gallery.html` with all assets and 3D models (`.glb` files) embedded as base64. The output requires no server or external dependencies to view.

## Viewing the output
The HTML generated by `generate_gallery.py` and embedded in the pub is available in `output/gallery.html`.

## Usage

To generate the .html gallery run:

```bash
python generate_gallery.py
```

Internet access is required at build time to download `model-viewer` and the Atkinson Hyperlegible Next font.


Open `gallery.html` in a browser. Navigate between models using the dropdown menu, the prev/next buttons, or the left/right arrow keys.

## Adding or modifying models

Edit `gallery_config.json`. Each entry:

| Field         | Required | Description                                                      |
|---------------|----------|------------------------------------------------------------------|
| `glb`         | yes      | Path to the `.glb` model file                                    |
| `label`       | yes      | Display name                                                     |
| `subtitle`    | no       | Additional label text (HTML allowed, e.g. `<i>Species name</i>`) |
| `bg_color`    | no       | Page background hex color (default `#ffffff`)                    |
| `model_color` | no       | Hex color applied to all model materials                         |
